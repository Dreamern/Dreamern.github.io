Relation-Aware Transformer for Portfolio Policy Learning

0 摘要

问题：Portfolio selection(PS)

关键点：

  • 如何表示投资组合中资产的非稳态价格系列

现有方法的不足：

  • 未能捕捉资产价格中复杂的sequential patterns
  • 多种资产间的价格相关性

本篇论文：

  • 使用强化学习方法
  • 提出了一种创新的Relation-aware Transformer(RAT)来处理现有不足
  • 提出了新的注意力机制，结合RAT来捕捉sequential patterns和资产相关性
  • 基于提取的序列特征，RAT能够通过新设计的杠杆操作来作出有利的投资组合策略

1 Introduction

由于以下两个挑战，现有的DNNs无法很好地提取价格序列特征：

  • 1.资产的价格序列遵循复杂的金融规律，使用现有DNNs难以捕捉
    • 既有短期趋势，又满足长期均值回归
    • 研究表明，RNNs(如LSTM和GRU)倾向于捕捉长期特征
  • 2.投资组合中的金融资产间通常具有复杂的相关性，并可能随着时间剧烈变化
    • 投资组合需要根据经济形势进行调整，但现有的DNNs只提取每种资产各自的时间序列特征，没有考虑资产间的相关性

近年来，注意力机制在序列建模上表现优异，如Transformer。Transformer不能直接用于PS，因为：

  • 不能捕捉局部context信息
  • 无法捕捉资产间的相关性

本文改进了Transformer，使其可以捕捉资产价格序列的局部sequential patterns和资产间的相关性。具体而言，将RAT作为PS问题的policy network。RAT是通过新提出的基于attention的模块来捕捉sequential patterns和资产相关性，并通过新设计的决策模块来作出有利的资产组合决定。通过强化学习来训练策略网络，RAT在PS性能上获得了明显提升。

主要贡献：

  • 提出了新的RAT方法for PS
  • 应用金融杠杆解决了现有的强化学习方法中的决策制定限制
  • 扩展实验表明模型效果优越

3 Relation-Aware Transformer

PS问题中，资产价格sequential patterns和资产间相关性都很重要。用RAT作为PS问题的策略网络。RAT需要捕捉这些信息并基于此做决策，因此我们扩展了标准的Transformer结构，使其relation-aware并且通过RL来处理任务。

3.1 General Architecture

如Fig.1，RAT遵循encoder-decoder架构。Encoder的输入是price series，Decoder的输入是local price context。

  • encoder: 提取序列特征，在PS问题中主要是sequential patterns和资产相关性。因此提出2个任务专用的attention模块
    • sequential attention layer
    • relation attention layer
  • decoder: 作出决策
    • 主要结构与encoder类似
    • 新增了decision-making layer，该层全面考虑了提取的特征，局部价格context以及上一时期的决策来选择资产组合
    • 其他模块与经典Transformer一致

3.4 Decision-making Layer

PS复杂度高，为RAT设计决策模块并不容易。现有的RL方法使用全连接和softmax，softmax不可避免地会要求资产的比例为正，即对于资产，，且。即使预测到资产未来价格会下降也无法避免损失。因此引入做空机制(short sale)。借入预测价格下跌的资产并卖出，然后买入预测价格上涨的资产。这样RAT可以作出更好的决策。

我们设计了新的杠杆操作，包含3个独立的softmax全连接决策层：

  • 第一个输出initial portfolio vector
  • 第二个输出short sale vector
  • 最后一个输出reinvestment vector

4 Related work

PS吸引了AI社区的大量注意力。基于Kelly原则，提出了许多PS算法，包括基于在线学习和RL的算法：

  • 基于online learning
    • 目的是最大化期望的log-return
    • 无法考虑序列特征，仅使用手工特征，如移动平均和随机技术指标，导致PS性能不稳定
  • 基于RL
    • 目的是最优化特定的函数并通过RL算法学习综合策略
    • 近期基于RL的研究很多使用深度学习来提取资产特征，在PS上的效果令人满意。
      • 最先进的如EIIE
      • 对抗深度RL算法
      • 利用CNNs或RNNs提取序列特征

Conclusion

本文：

  • 展示了一种创新的RAT来学习投资组合策略
  • RAT可以同时捕捉sequential patterns和资产相关性，并作出有利的投资组合决定
  • 通过强化学习来训练策略网络（如RAT），我们的方法带来了明显的性能提升。在一些扩展实验中也表现出了优越性
  • 根据实验结果，我们认为
    • 注意力机制可以认为是PS的主要方案
    • 杠杆操作是基于强化学习的PS方法的收益关键
