BEIT: BERT Pre-Training of Image Transformers

摘要

我们介绍了自监督视觉表示模型BEIT，即Bidirectional Encoder representation from Image Transformers.受NLP领域的BERT启发，我们提出了一种masked image modeling(MIM)来预训练vision Transformers。具体而言，每张图(src_img)像有2个视角：

  • image patches(16x16 pixels)，下图有16个patch，实验中一张图分为196个patch(224/16, 224/16)

  • visual tokens(discrete tokens)（根据图像学到的单词，即将图像向量化）

实验方法（结合下图）：

  • 路线一（横向，visual tokens）：原始图像经过tokenizer（VQ-VAE）转换为visual tokens
  • 路线二（纵向，image patches）：原始图像分割成image patches，随机mask一些patches（如40%）并将patch序列展平，然后输入到backbone Transformer，让Transformer来预测visual tokens

预训练的目标：基于破损（掩码）的image patches恢复原始的visual tokens

在预训练BEIT后，通过在预训练的encoder上添加task层来直接调优下游任务的模型参数

结论：在图像分类和语义分割上实现了有竞争力的结果。

Introduction

Transformer在cv领域取得了很好的表现。然而，经验研究表明ViT需要比CNN更多的训练数据。为了解决data-hungry问题，自监督预训练是一个获得大规模图像数据的有前景的解决方案。

同时，BERT在NLP领域获得了巨大成功。其masked language modeling(MLM)任务首先随机掩盖文本中的一部分tokens，然后基于Transformer对破损文本的编码结果来修复被掩码的tokens。受BERT启发，我们转向了用降噪自编码来预训练ViT的想法，这在视觉领域还没有被很好地研究。对图像数据直接应用BERT风格的预训练是一个挑战。首先，对于vision Transformer的输入单元没有预先存在的词汇表，即：image patches。因此，我们不能简单地使用softmax分类器来预测所有可能的masked patches候选。相比之下，语言词汇表（如words和BPE）定义明确，简化了自编码预测。一个直接的替代方案是将任务视为回归问题，它预测masked patches的原始像素。然而，这种像素级恢复任务往往会在训练前的短程依赖性和高频细节上浪费建模能力。我们的目标是克服ViT预训练中的上述问题。

在本工作中，我们引入了一种自监督视觉表示模型BEIT，即Bidirectional Encoder representation from Image Transformers(来自ViT的双向编码器表达)。受BERT启发，我们提出了一个预训练任务，即masked image modeling（MIM，掩码图像建模）。

如图1所示，MIM从2个方面对待每一张图片：image patches和visual tokens。我们将图像分割为patches，patch序列作为backbone Transformer的输入表达。此外，我们将图像tokenize成离散的visual tokens，visual tokens是从discrete VAE的隐编码获得的。在预训练期间，我们随机地屏蔽一定比例的图像patches，并且将损坏的patch序列输入到Transformer。该模型学习恢复原始图像的visual tokens，而不是被掩码的patches的原始像素。

我们通过自监督学习，然后在两个下游任务上微调预训练的BEIT，即：图像分类和语义分割。实验结果表明，BEIT的性能优于从头训练和已有的强自监督模型。此外，BEIT是对有监督预训练的补充。BEIT的性能可以通过使用ImageNet标签进行中间微调来进一步提高。消融研究表明，我们提出的技术对BERT类型的图像数据预训练的有效性至关重要。除了性能之外，收敛速度和微调稳定性的提高降低了端任务的训练成本。此外，我们还证明了自监督BEIT可以通过预训练学习合理的语义区域，释放图像中包含的丰富的监督信号。

我们的贡献概述如下：

  • 我们提出了一个掩码图像建模任务，以自监督的方式预训练ViT，并从变分自编码器（VAE）的角度给出了理论解释。
  • 我们对BEIT进行预训练，并对图像分类和语义分割等下游任务进行大量微调实验。
  • 我们展示了自监督BEIT的自注意机制学会了区分语义区域和对象边界，尽管没有使用任何人工注释。

2 Methods

给定输入图像，BEIT将其编码为上下文化的向量表示。如图1所示，BEIT由自监督模型中的掩码图像建模（MIM）任务进行预训练。MIM的目标是基于编码向量恢复被掩码的图像patch。对于下游任务（如图像分类和语义分割），我们在预训练的BEIT上添加task layers，并在特定数据集上微调参数。

预训练目标是最大化log-likelihood of the correct visual tokens given the corrupted image: 是training corpus(图像数据集)，是掩码位置，是根据掩码后的破损patches。

BEIT不是随机选择掩码位置，而是使用blockwise masking。如在算法1中总结的，每次掩码一个block内的图像patches。对于每个block，我们将最小patches数设置为16。然后随机选择掩码block的纵横比。重复上述两个步骤，直到获得足够多的掩码patches，即，其中是图像patches的总数，并且0.4是掩码比。

MIM任务主要受masked language modeling启发，这是自然语言处理中最成功的一个训练目标。此外，blockwise(or n-gram)掩码也广泛应用于BERT-like模型。然而，在视觉预训练中直接使用进行像素级auto-encoding(即修复掩码patches的像素)会导致模型专注于短程依赖性和高频细节。BEIT通过预测离散的visual tokens克服了上述问题，它将细节进行了高层抽象。3.3节的消融研究表明，我们提出的方法明显优于进行像素级auto-encoding。

2.5 Pre-Training Setup

BEIT的网络架构遵循ViT-Base的网络架构，以便进行公平的比较。我们使用一个12层的Transformer，hidden size为768，有12个attention head。前馈网络的中间size为3072。我们使用默认的 input patche size。我们直接借用DALL-E训练的image tokenizer。visual tokens的词汇量为8192。

我们在ImageNet-1K的训练集上预训练BEIT，该训练集包含大约1.2M个图像。我们的增强策略包括随机裁剪大小、水平翻转、颜色抖动。请注意，我们不使用标签进行自监督学习。我们在实验中使用了的分辨率。因此，输入被分割成个图像块，以及相同数量的visual tokens。我们随机掩码最多75个patches（约总图像patch数的40%）。

预训练运行大约500k steps（800个epochs），batch size为2k。采用Adam()进行优化。学习速率被设置为，with warmup of 10 epochs和余弦学习率衰减。权重衰减为0.05。我们采用随机深度，速率为0.1，并禁用dropout。500k的训练steps需要大约五天的时间，使用16块英伟达Telsa V100 32GB GPU卡。

我们发现适当的初始化对于Transformer的稳定性非常重要，尤其是对于大规模的预训练。我们首先在一个小范围内随机初始化所有参数，例如。然后，对于第个Transformer层，我们用重新调整自注意力模块（每个子层内最后的linear projection层）和前馈网络的输出矩阵。

2.6 Fine-Tuning BEIT on Downstream Vision Tasks

在预训练BEIT之后，我们在Transformer上附加一个task layer，并微调下游任务的参数。本文以图像分类和语义分割为例进行了研究。使用BEIT在其他视觉任务上利用pre-training-then-fine-tuning paradigm范式是很简单的。

Image classification

对于图像分类任务，我们直接采用一个简单的线性分类器作为任务层。具体来说，我们使用平均池化来聚合表达，并将全局表达提供给softmax分类器。类别概率的计算公式为softmax()，是第个图像patch的最终编码向量，是一个参数矩阵，是label数。我们通过更新BEIT和softmax分类器的参数来最大化标签数据的似然。

Semantic segmentation

对于语义分割，我们遵循SETRPUP中使用的任务层。具体地说，我们使用预先训练的BEIT作为backbone encoder，并结合几个反卷积层作为decoder来产生分割。该模型也是端到端微调，类似于影像分类。

Intermediate fine-tuning

在自监督预训练之后，我们可以进一步在数据丰富的中间数据集上训练BEIT（如我们工作中的ImageNet-1K），然后在目标下游任务上微调模型。这种中间微调是NLP中BERT微调的常见实践。我们直接遵循BEIT的方法。

Conclusion

我们介绍了用于ViT的自监督预训练框架，在下游任务上实现了不错的微调结果，如图像分类和语义分割。

我们提出的方法对于BERT-like预训练（auto-encoding with masked input）能很好地工作很关键。

我们还展现了语义领域中不借助任何人工标注的数据而自动获取知识的有趣属性。

未来希望在数据集和模型方面上加大BEIT预训练的规模。

更重要的是，通过更一般的方式，基于文本和图像共享的架构和目标来开展多模态预训练。
